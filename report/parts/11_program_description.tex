\chapter{Описание программы}
Результатами разработки являются PAM модуль \emph{pam\_twofactor\_auth} и программа \emph{aes\_generate}.

\section{Сборка}
\label{compilation}
Для сборки разработанного программного продукта необходимо выполнить следующие шаги:
\begin{enumerate}
\item клонировать репозиторий~\cite{repo};
\item создать директорию для бинарных файлов (далее предполагается, что эта директория называется \t{build} и находится в корне репозитория;
\item запустить команду \t{ccmake ..};
\item прописать нужный тип сборки (например, \emph{Debug} или \emph{Release}), сконфигурировать \emph{CMake} и сгенерировать им \t{Makefile};
\item запустить команду \t{make}.
\end{enumerate}
Если сборка выполнится успешно, то в директории \t{build} будут бинарные файлы \t{pam\_twofactor\_auth.so} и \t{aes\_generate}.

\section{Поддержка разных ОС}
Разработанный продукт поддерживает 2 операционных системы:
\begin{itemize}
\item МСВС 3.0;
\item Ubuntu 12.04 и новее.
\end{itemize}

Следует обратить внимание на то, что в операционной системе Ubuntu изначально нет утилиты \emph{ccmake}, поэтому для сборки проекта необходимо
предварительно выполнить команду \t{sudo apt-get install cmake-curses-gui}.
Кроме того, для клонирования репозитория необходимо установить \emph{git}. Альтернативный вариант: вместо первого шага сборки скачать архив с репозиторием
и распаковать его.
